thutils
=======

About
=====

Personal python utility library

Dependencies
============

Python 2.5+ or 3.3+

-  `python-magic <https://pypi.python.org/pypi/python-magic/>`__
-  `six <https://pypi.python.org/pypi/six/>`__
-  `voluptuous <https://pypi.python.org/pypi/voluptuous/>`__

Installation
============

::

    pip install thutils
